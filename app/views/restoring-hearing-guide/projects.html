Using light to improve cochlear implants

Project
Using light to improve cochlear implants
Researcher
Professor Rachael Richardson
Where
Bionics Institute
Melbourne
Australia

If you are profoundly deaf, cochlear implants can restore hearing by working in place of hair cells that have been lost. They generate electrical signals that activate auditory nerve cells which carry signals to the brain, which allows you to hear. But it can still be difficult to hear in some situations especially when there is a lot of background noise.

Professor Richardson is working to improve cochlear implants by using light, instead of electrical signals, to activate the hearing nerve cells. This could help people with implants hear with a much better and more natural quality of sound.
